# FRONTEND-BASICS

This is my first web development task in PBL at Linkcode Technologies.

# PROJECT DESCRIPTION 

Learn the fundamentals of web development by setting up the development environment, creating a static webpage using HTML, applying basic CSS styling, and pushing the project to GitHub.


# SUBTASK

1. Set up VS Code, install Live Server extension, create first HTML file
2. Build a static webpage: heading, paragraph, image, hyperlink, ordered & unordered list
3. Apply basic CSS: background colour, font-family, font-size, text colour
4. Push project to GitHub (first commit) — learn add, commit, push

# TECHNOLOGIES USED

- HTML5
- CSS3
- VS Code
- Git & GitHub
